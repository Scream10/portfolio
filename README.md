## Outils de développement

* ReactJS
* Sass
* GSAP
* ScrollMagic
* HTML / CSS
* Javascript
* Webpack

## Lancer localement

* npm install
* npm start
* Dispo sur le serveur [http://localhost:3000](http://localhost:3000)

Ou

* yarn install
* yarn start
* Dispo sur le serveur [http://localhost:3000](http://localhost:3000)
